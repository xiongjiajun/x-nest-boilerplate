## Description
可自动引入Controller、Provider和Module，开发者只需关注业务

### 支持自动生成代码
yarn gencode &lt;new-work&gt; [-f] [extends &lt;parent-work&gt;]

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
